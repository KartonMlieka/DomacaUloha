Čo je GitHub?

GitHub je webová platforma určená pre správu verzií a spoluprácu na softvérových projektoch. Slúži na uľahčenie sledovania zmien v kóde, diskusie o týchto zmenách a spoluprácu viacerých vývojárov na jednom projekte. GitHub používa systém kontroly verzií nazývaný Git, ktorý umožňuje ukladanie a spravovanie rôznych verzií kódu.

Načo sa používa?

Vytváranie vzdialených, verejne dostupných úložísk v cloude. Úložiská, predstavujú súbory projektu kódovania a uchovávajú históriu revízií každého súboru. Tieto uložiská sa dajú naklonovať na vaše zariadenie, na ktorom môžte robiť zmeny. A potom neskôr, svoje zmeny vložiť spať do uložiska, kde sa pre verejnosť aktualizujú.

Ako začať?

  • Zaregistruj sa na GitHube
  • Nainštaluj GIT
  • Prepoj svoj účet s GIT-om
  • Vytvoj kód na svojom zariadený
  • Sprav uložisko na GitHube
  • Vlož kód na GitHub
